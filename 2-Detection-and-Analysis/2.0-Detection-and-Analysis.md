# Detection and Analysis

This phase focuses on identifying, investigating, and understanding security events and potential threats within your environment. It covers:

- [**SIEM and Logging** ](./SIEM-and-Logging.md): Aggregating and analyzing logs from endpoints, servers, and network devices to detect suspicious activity.
- [**Monitoring Sensors**](./Monitoring-Sensors.md): Deploying host, network, and OT sensors for real-time visibility and alerting.
- [**Digital Forensics**](./Digital-Forensics.md): Acquiring and analyzing forensic evidence from memory, disk, and network sources.
- [**Detection Tools Reference**](./2.1-Tools.md): Testing and improving detection coverage using adversary simulation tools.
